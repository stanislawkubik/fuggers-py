# `fuggers_py.market`

Retired public namespace.

This path is no longer part of the public API.

Use the first-layer packages instead:

```python
from fuggers_py.curves import YieldCurve
from fuggers_py.vol_surfaces import VolatilitySurface
from fuggers_py.bonds import BondQuote
from fuggers_py.rates import IndexConventions
```

The old market-state and market-data helpers now live under the internal
`_market` package.
